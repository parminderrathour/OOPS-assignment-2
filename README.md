## 📖 About

This repository contains the solution to **Assignment 2** of the Object-Oriented Programming course. The assignment dives into core OOP concepts including **structures**, **classes**, **access specifiers**, **member functions**, **scope resolution**, and **namespaces** in C++.

---

## ✅ Questions Covered

| Q# | Topic | Description |
|---|---|---|
| Q1 | Structures | Create a `Student` struct with fields like name, roll no., degree, hostel, and CGPA; add, update, and display details |
| Q2 | Classes & Access Specifiers | Implement a `Student` class using `private` and `public` access specifiers for data encapsulation |
| Q3 | Private & Public Functions | Demonstrate calling a private member function through a public member function |
| Q4 | Class with Methods | Create a `Rectangle` class with `getdata()` and `calculateArea()` member functions |
| Q5 | Complex Number Addition | Implement a `Complex` class with `setComplex()`, `displayComplex()`, and `add()` methods |
| Q6 | Scope Resolution Operator | Distinguish between local and global variables using the `::` scope resolution operator |
| Q7 | Namespaces | Create two separate namespaces and access their members using the `::` operator |

---

## 🧠 Concepts Used

- **Structures (`struct`)** — grouping related data fields
- **Classes & Objects** — defining blueprints and creating instances
- **Access Specifiers** — `private` and `public` for data encapsulation
- **Member Functions** — functions defined inside a class
- **Scope Resolution Operator (`::`)** — distinguishing local vs. global variables
- **Namespaces** — avoiding name conflicts using named scopes
- **Object as Return Type** — returning objects from functions (Complex addition)

## ▶️ How to Run

1. **Clone this repository:**
   ```bash
   git clone https://github.com/parminderrathour/OOPS-assignment-2.git
   cd OOPS-assignment-2
   ```

2. **Compile the file using g++:**
   ```bash
   g++ parminder_ass2.cpp -o output
   ```

   ```bash

   ```
